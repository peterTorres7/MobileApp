## Setup
🛠

1. Clone the repository

```Bash
git clone https://github.com/peterTorres7/MobileApp.git
```
2. Change the working directory

```Bash
cd MobileApp
```

3. Add Environment Variable path of this directory

4. Run the app using terminal
